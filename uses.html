Uses / Developer Setup

This page lists my current daily driver setup for coding, studying, and technical note-taking.

Hardware

  • Main Laptop: repairable and upgrade-friendly build focus
  • Backup Laptop: lightweight travel option for classes
  • Keyboard: mechanical keyboard with tactile switches
  • Notes: e-ink style digital notebook workflow

Software Stack

  • Editor: VS Code / JetBrains tools for coursework and projects
  • Languages: C++, Java, Python, and front-end basics
  • Version Control: Git + GitHub for portfolio and class projects
  • Design: Onshape for 3D models and prototyping

Focus Setup

  • Fonts: clean coding fonts with clear symbol readability
  • Theme: mixed dark-first editor theme with high contrast comments
  • Terminal: simple prompt and larger text for quick scanning
  • Workflow: short tasks, commit often, and keep notes while debugging

Inspiration

I enjoy the clarity and personality of maggieappleton.com and the practical setup style from wesbos.com/uses.
